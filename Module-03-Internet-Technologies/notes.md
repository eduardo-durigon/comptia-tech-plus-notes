# 3.1 🌍 What Is the Internet?

The **Internet** is a huge collection of interconnected networks that share information.

- It connects devices globally
- It uses hardware, software, and protocols
- It is decentralized (no single owner)

### Key Idea:
The Internet is the infrastructure.  
The **World Wide Web (WWW)** is a service that runs on the internet.

---

## 🧠 Protocols

Protocols are sets of rules that allow devices to communicate.

### High-Level Protocols
- Email
- File downloads
- Web browsing

### Low-Level Protocols
- Support high-level protocols
- Most important: **TCP/IP**

### 🔹 TCP/IP
- Transmission Control Protocol / Internet Protocol
- Core protocol of the internet
- Ensures data is addressed and delivered properly

# 🌐 Web Basics

## HTTP & HTTPS

- **HTTP** (HyperText Transfer Protocol) → Transfers web data
- **HTTPS** → Secure version of HTTP
  - Uses TLS encryption
  - Protects data in transit

## HTML

- HyperText Markup Language
- Used to structure webpages

## CSS

- Cascading Style Sheets
- Controls appearance and layout of webpages

# 🔐 Safe Internet Browsing

## Certificates

Websites use certificates to prove identity.

Browsers check:
1. Is it from a trusted Certificate Authority (CA)?
2. Is it valid and not expired?

HTTPS indicates secure communication.

---

## Browser Privacy Features

- Block third-party cookies
- Incognito / Private mode
- Pop-up blockers
- Extensions (must come from trusted sources)

⚠ Private mode does NOT hide your IP address.  
For that, you need a VPN or proxy.

# ⚡ Internet Speed & Performance

## Speed
Measured in:
- Mbps (Megabits per second)
- Gbps (Gigabits per second)

# 🌐 Types of Internet Connections

---

## 1️⃣ Cable Internet

- Uses coaxial cables
- Fast and widely available

### Equipment:
- Coaxial cable
- Modem
- Router

### Speeds:
- 10 Mbps to 2+ Gbps
- Latency: 10–50 ms

---

## 2️⃣ DSL (Digital Subscriber Line)

- Uses telephone lines
- Older broadband technology

### Types:
- ADSL (Asymmetric)
- SDSL (Symmetric)

### Speeds:
- 5–30 Mbps (sometimes up to 100 Mbps)
- Latency: 20–50 ms

Signal quality depends on:
- Distance from provider hub
- Wire quality

---

## 3️⃣ Fiber Internet

- Uses fiber optic cables
- Transmits data using light

### Structure:
- Glass or plastic core
- Reflective cladding
- Protective outer layers

### Connectors:
- SC (Subscriber Connector)
- LC (Lucent Connector)

### SFP Transceivers
- Small form-factor pluggable modules
- Used in fiber networking equipment

### Speeds:
- 100 Mbps to 25 Gbps
- Latency: 10–25 ms
- Symmetrical speeds (equal upload & download)

### FTTC (Fiber to the Curb)
- Fiber to street cabinet
- Final stretch via telephone wiring (VDSL)

---

## 4️⃣ Wireless Internet (Cellular)

- Uses cellular towers
- Same network as mobile phones

### Equipment:
- Modem
- Router

### Speeds:
- 40–300 Mbps
- Latency: 1–30 ms

### 5G:
- Download: 40–300 Mbps
- Upload: 10–50 Mbps
- Latency: ~30 ms

6G is developing but not widely available.

---

## 5️⃣ Satellite Internet

- Used in rural areas
- Requires satellite dish

### Providers:
- HughesNet
- Viasat
- Starlink

### Speeds:
- 12–250 Mbps download
- ~3 Mbps upload

### Limitations:
- High latency (100+ ms)
- Weather affects connection
- Data caps

---

## 6️⃣ Fixed Wireless Internet

- Uses antennas
- Requires line-of-sight to tower
- Can handle bad weather better than satellite

# 🔐 Security Concepts (From Module 3 Notes)

## CIA Triad

### 1️⃣ Confidentiality
- Keep data private
- Encryption protects data

### 2️⃣ Integrity
- Ensure data is not altered
- Hashing verifies integrity
- Examples: MD5, SHA

### 3️⃣ Availability
- Data must be accessible
- Protect against DoS attacks

---

## 🔑 Authentication & Authorization

### Authentication
Proving identity:
- Something you know (password)
- Something you have (token)
- Something you are (biometrics)

### Authorization
- What you are allowed to do

### Least Privilege
- Users get only necessary access
