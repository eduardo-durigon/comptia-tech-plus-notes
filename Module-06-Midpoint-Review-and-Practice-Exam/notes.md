# Concepts Reviewed During the Practice Exam

## Networking and Internet Concepts

- **IPv6 address size:** 128 bits  
- **IPv4 address size:** 32 bits  
- **Maximum value in a byte:** 255  
- **Hexadecimal digits represent:** 4 bits  
- **MAC address hexadecimal pairs:** 6  

---

## Wireless Networking

- **First Wi-Fi standard allowing multiple channels:** 802.11n  
- Wireless internet access can generally be used **anywhere within the home network coverage area**.

---

## Network Infrastructure

- **DSL drawback:** Signal attenuation  
- **DSL uses:** Telephone lines  
- **Ethernet cable wires:** 8 wires  

---

## Security Concepts

### Firewall Types

- **Software firewall** protects a single workstation or server.

---

### Anonymous Browsing

Tools used to browse anonymously include:

- VPN
- Proxy servers

---

### Security Monitoring

- **Real-time antivirus scanning** constantly monitors a system for threats.

---

### Defense in Depth

A layered security approach used to protect systems through multiple security controls.

---

### Network Administration

- **LAN networks** are typically managed by a **network administrator**.

---

## System and Network Concepts

### Intranet

A **private internal network** used within an organization.

---

### Virtualization

Virtual services help administrators by:

- **Consolidating resources**
- Running multiple systems on fewer physical machines

---

### NAT (Network Address Translation)

Used to translate internal private addresses to public addresses for internet communication.

---

## Operating System and Permissions

- **Linux file permissions** are represented using **octal numbers**.

## Data Representation

ASCII converts **text characters into binary code** that computers can process.
